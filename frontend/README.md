# KitchAI Landing

Landing page y frontend base de KitchAI, construida con Next.js, TypeScript y Tailwind CSS.

## Requisitos

- Node.js 20+
- pnpm

## Instalacion

```bash
pnpm install
```

## Desarrollo

```bash
pnpm dev
```

La aplicacion queda disponible en `http://localhost:3000`.

## Scripts

- `pnpm dev`: inicia el entorno de desarrollo
- `pnpm build`: genera el build de produccion
- `pnpm start`: levanta el build generado

## Estructura

```text
app/          rutas y paginas con App Router
components/   componentes publicos, dashboard y UI
hooks/        hooks reutilizables
lib/          utilidades y datos mock
public/       assets estaticos
styles/       estilos adicionales
```
